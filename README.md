# Penggabung Nama (Name Combiner)

Aplikasi Flutter sederhana yang memungkinkan Anda menggabungkan dua nama menjadi berbagai variasi dan kombinasi unik. Sangat cocok digunakan untuk mencari nama panggung, nama pasangan (ship name), atau sekadar bersenang-senang.

## Fitur
- Input dua nama yang berbeda.
- Menghasilkan berbagai kombinasi potongan nama.
- Desain antarmuka responsif yang bekerja dengan baik di perangkat seluler, tablet, web, dan desktop.

## Stack Teknologi
- **Framework:** Flutter
- **Bahasa:** Dart

## Cara Menjalankan Aplikasi
1. Pastikan Flutter SDK telah terinstal di sistem Anda.
2. Clone atau unduh repositori ini.
3. Jalankan `flutter pub get` di terminal untuk mengambil semua dependensi.
4. Jalankan aplikasi menggunakan perintah `flutter run` atau jalankan melalui IDE pilihan Anda.
